MY PROJECTS

Real-Time Currency Converter

A Real-Time Currency Converter allows users to input an amount in one currency and see the equivalent amount in another. The HTML provides input fields for selecting currencies and entering the amount, while CSS styles the interface with a modern look. JavaScript (not CSS alone) is used to fetch live exchange rates, calculate conversions, and update the results in real-time.

To-Do list

A To-Do List application lets users add, check off, and remove tasks, helping them stay organized. HTML is used to create a form for adding tasks and listing items, while CSS styles the layout with a clean, minimalist design. With JavaScript, users can mark tasks as completed, dynamically add new tasks, and delete tasks as needed.

BMI (Body Mass Index)

A BMI (Body Mass Index) Calculator takes user input for weight and height to calculate BMI, providing an indicator of healthy body weight. The HTML includes input fields and a button to start the calculation. CSS styles the calculator with a professional, easy-to-read design. JavaScript is required to perform the BMI calculation and display the result based on the input values.
